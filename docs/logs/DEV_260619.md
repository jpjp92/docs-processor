# DEV_260619

## Summary

설정에서 불필요한 "백엔드 주소" 항목을 제거하고, 1,500줄이 넘던 `components/pdf-workspace.tsx` 모놀리스를 단계적으로 리팩토링했다.

- 백엔드 주소 설정 제거 (통합 Next.js 구조에서 동작하지 않는 잔재)
- Phase A: 비-UI 로직(타입·상수·순수 헬퍼·리포트 빌더)을 `lib/pdf-workspace/`로 추출
- Phase B: 큰 하위 컴포넌트 분리
- Phase C: 도메인별 커스텀 훅으로 상태 정리

## Backend Setting Removal

- 출처: 원본 `Ref/pdf-workspace`(정적 HTML + 별도 Express `server.js`) 구조의 잔재.
- 현재는 단일 Next.js 앱이라 프론트/API가 같은 origin → `backend` 필드는 불필요하고, 교차 origin은 CORS 미설정으로 동작하지도 않았다.
- 제거 범위: `ai`/`draftAi` 상태 필드, 설정 UI 입력칸, 저장 핸들러, fetch 3곳(`/api/config`·`/api/analyze`·`/api/summarize`)을 상대경로로 변경.

## Refactor — Phase A (순수 추출)

`components/pdf-workspace.tsx`에서 동작 변화 없이 비-UI 로직을 분리.

| 모듈 | 내용 |
|------|------|
| `lib/pdf-workspace/types.ts` | Provider, Clip, Pdf* 타입 + `AnalysisExport` + Window 전역 선언 |
| `lib/pdf-workspace/constants.ts` | PDFJS URL, PROVIDER_INFO, PRESETS, 기본 분석 프롬프트 |
| `lib/pdf-workspace/format.ts` | escapeHtml, renderMarkdown, downloadTextFile, safeFileStem |
| `lib/pdf-workspace/pdfjs.ts` | loadPdfJs |
| `lib/pdf-workspace/report.ts` | buildReportHtml (인라인 HTML 리포트 템플릿 분리) |

- `downloadAnalysisHtml`은 140줄 인라인 템플릿 → `buildReportHtml(buildAnalysisExport())` 4줄로 축소.
- `pdf-workspace.tsx`: 1,524 → 1,177줄.

## Refactor — Phase B (컴포넌트 분리)

큰 하위 컴포넌트를 별도 파일로 분리. 동작 변화 없음(프레젠테이션 컴포넌트화).

| 컴포넌트 | 비고 |
|----------|------|
| `components/clip-card.tsx` | 기존 ClipCard 이동 |
| `components/doc-summary-panel.tsx` | 문서 요약 패널. `SummaryState` prop |
| `components/settings-modal.tsx` | 설정 모달. 파생값(currentProviderInfo 등)은 내부 계산, 핸들러는 props |

- 공유 타입 `SummaryState`, `AiSettings`를 `lib/pdf-workspace/types.ts`에 추가.
- `pdf-workspace.tsx`: 1,177 → 915줄.

## Refactor — Phase C (커스텀 훅)

도메인 상태/로직을 훅으로 묶어 컴포넌트의 useState 평면 나열을 줄임.

| 훅 | 책임 |
|----|------|
| `hooks/use-summary.ts` | 요약 상태 + 전체 텍스트 수집 + 스트리밍 요약(runSummary/resetSummary) |
| `hooks/use-ai-settings.ts` | ai/draftAi, serverKey/disabled providers, 설정창 open/save/close + /api/config |

- `pdf-workspace.tsx`: 915 → 800줄 (시작 1,524줄 대비 약 47% 감소).
- usePdfDocument(렌더링·ref 다수)는 결합도가 높아 이번 범위에서 제외, 추후 별도 진행.
